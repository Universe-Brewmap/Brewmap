# 바나프레소 신규 입점 입지 추천 시스템

> 기존 바나프레소 매장의 입지 패턴을 분석하여 신규 창업 후보지를 추천하는 **데이터 기반 입지 분석 시스템**

**팀원:** 김단하 · 김민정 · 조승아  |  **기간:** 2026.01.19 ~ 2026.03.06

---

##  프로젝트 개요

대부분의 예비 창업자는 데이터 분석 없이 직관과 경험에 의존해 입지를 결정한다. 본 프로젝트는 서울 내 바나프레소 기존 매장 데이터를 분석해 이상적인 입지 프로파일을 도출하고, 신규 매물을 점수화하여 최적 입점 후보지를 추천한다.

## 점수화 구조

```
최종 점수 (100점) = 입지 점수 (75점) + 비용 리스크 점수 (25점)
```

### 입지 점수 (75점)

| 카테고리 | 가중치 | 주요 변수 |
|--------|------|---------|
| 유동인구 | 38% | 점심·저녁·기타 시간대 유동인구 |
| 접근성 | 37% | 지하철거리·버스거리·대로변·횡단보도·층수 |
| 주변 시설 | 8% | 학원·병원·학교·기업 수 |
| 경쟁 강도 | 17% | 바나프레소·고가카페·저가카페·기타카페 수 |

- 기존 매장 운영기간 가중치 적용 (3년↑: 3.0 / 1~3년: 1.0 / 1년↓: 0.5) → 이상적 입지 프로파일 산출
- lower_better 변수(거리·경쟁)는 부호 반전 후 MinMaxScaler 정규화

### 비용 리스크 점수 (25점)

- **월세 점수 (10점):** 행정동 평균 대비 상대적 저렴함
- **자본금 충당 점수 (15점):** 자본금 1.2억 기준 실질 충당 가능성

## 머신러닝 모델 (Logistic Regression)

소표본(59개) 환경을 고려해 과적합 방지와 해석 가능성 확보를 위해 Logistic Regression 선택.

| 항목 | 내용 |
|-----|------|
| 입력(X) | 16개 피처 (MinMax 정규화 후) |
| 타깃(y) | success (3년 이상=1, 미만=0) |
| 검증 | Stratified K-Fold (k=5) |
| 정규화 | L2, 최적 C=0.01 |
| AUC | 0.767 (교차검증 기준) |
| Accuracy | 0.763 (훈련 데이터 기준, 참고용) |

> ⚠️ 훈련 데이터 기준 지표는 참고용. 신뢰 가능한 성능은 교차검증 AUC 0.767 기준으로 판단.

## 추천 결과

```
전체 매물 → [1차] recom_score 상위 50% 필터 → [2차] success_prob 내림차순 정렬 → TOP 20
```

TOP 20 추천 매물 공통 특성:
- 1층 위치
- 대로변 접근 (is_road = 1)
- 지하철역 300m 이내
- 점심 유동인구 상위권
- 자본금 1.2억 충당 가능
- 자사 경쟁 매장 없음

## 핵심 인사이트

1. **오피스 상권 중심** — 점심 유동인구가 가장 중요한 변수, 직장인 밀집 지역 출점 패턴 확인
2. **가시성·접근성 핵심** — 대부분 매장이 대로변 + 1층, 지하철역 평균 280m 이내
3. **소비력 기반 전략** — 고가 카페 多 = 소비력 높은 상권 신호, 자사 경쟁 최소화
